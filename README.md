🐍 Belajar Python Dasar

* **01 Python Basic:** Mengenal variabel, tipe data, dan operasi dasar.
* **02 Python Intermediate:** Belajar List, Dictionary, dan Tuple.
* **03 Python Foundation:** Mempelajari fungsi (Function) dan logika program.
* **04 Database Project:** Implementasi Python dengan penyimpanan data.
* **05 Mini Project:** Penerapan semua materi ke dalam aplikasi sederhana.
